Verwendung

  1. Erzeugen Sie ein Preset mit Hilfe des jQueryAdmin Admin Tools, indem Sie dort "Plugins" => "FancyBox" markieren
  2. Speichern Sie dieses Preset im jQueryAdmin presets Unterverzeichnis (z. B. unter dem Namen "fancybox")
  3. Integrieren Sie folgenden Droplet-Aufruf z.B. in Ihr Template oder eine WYSIWYG-Sektion:
    [[jQueryInclude?preset=<Presetname>]]
    (Ersetzen sie <Presetname> durch den oben vergebenen Namen, also etwa "fancybox")
  4. Fügen Sie das rel="<groupname>" Attribut bei den Links hinzu, die auf das Bild in Originalgröße verweisen (wobei "<groupname>" ein beliebiger Name ist; vorgegeben ist "grouped_elements")
    Beispiel:

    <a class="grouped_elements" rel="group1" href="image_big_1.jpg">
    <img src="image_small_1.jpg" alt=""/>
    </a>
Hinweis: Wenn Sie standardmäßig einen anderen Gruppennamen verwenden möchten, können Sie die Datei default.preset im Unterverzeichnis plugins/FancyBox editieren, um dort den Gruppennamen zu ändern. "grouped_elements" ist der Standardwert.
