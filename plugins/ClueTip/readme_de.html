Verwendung

  1. Erzeugen Sie ein Preset mit Hilfe des jQueryAdmin Admin Tools, indem Sie dort "Plugins" => "ClueTip" markieren
  2. Speichern Sie dieses Preset im jQueryAdmin presets Unterverzeichnis (z. B. unter dem Namen "cluetip")
  3. Integrieren Sie folgenden Droplet-Aufruf z.B. in Ihr Template oder eine WYSIWYG-Sektion:
    [[jQueryInclude?preset=<Presetname>]]
    (Ersetzen sie <Presetname> durch den oben vergebenen Namen, also etwa "cluetip")
  4. Fügen Sie das class="<classname>" Attribut bei den Links hinzu, die den clueTip bekommen sollen, wobei <classname> einer der folgenden Werte sein kann:
    • clfragment
    • cllocal
    • cltitle

    Beispiel: <a href="#local" class="cllocal" rel="#local" >link</a>
    Folgende Möglichkeiten sind vordefiniert:

    Den Inhalt des title Attributes verwenden
    <a class="cltitle" href="#" title="|Houdini was an escape artist.|He was also adept at prestidigitation.">Houdini</a>
    Demo
    Eine externe HTML-Datei als clueTip verwenden
    <a class="clfragment" href="fragment.html" rel="fragment.html">link</a>
    Demo
    Den Inhalt eines Blocks innerhalb der aktuellen Seiten verwenden
    <a class="cllocal" href="#myid" rel="#myid">link</a>
    ...later...
    <div id="myid">Dieser Text wird als clueTip angezeigt.</div>
    Demo
    Dieser Text wird als clueTip angezeigt.
    In dieser Variante wird automatisch die Option "sticky" verwendet.


    Sie können die Optionen ändern oder neue hinzufügen, indem Sie die Datei default.preset im Plugin-Verzeichnis anpassen.
