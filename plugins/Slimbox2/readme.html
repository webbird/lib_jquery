Verwendung

Einbinden der Dateien


Folgende Dateien müssen im <head>-Bereich eingebunden werden:

<script type="text/javascript" src="{CAT_URL}/modules/lib_jquery/plugins/Slimbox2/jquery-slimbox2-min.js"></script>
<link rel="stylesheet" href="{CAT_URL}/modules/lib_jquery/plugins/Slimbox2/slimbox2.css" type="text/css" />

Dies kann direkt im Template oder mit Hilfe einer headers.inc.php Datei erfolgen.

Lightbox aktivieren


Einzelnes Bild
<a href="images/image-1.jpg" rel="lightbox" title="my caption">image #1</a>

Eine Gruppe von Bildern
Hierzu dem rel "lightbox" einen eindeutigen Namen anhängen, hier z.B. "-cats":
<a href="images/image-1.jpg" rel="lightbox-cats">image #1</a>
<a href="images/image-2.jpg" rel="lightbox-cats">image #2</a>
<a href="images/image-3.jpg" rel="lightbox-cats">image #3</a>

Es ist kein weiteres Javascript notwendig, nur rel="lightbox" benutzen und fertig!
