Verwendung

  1. Erzeugen Sie ein Preset mit Hilfe des jQueryAdmin Admin Tools, indem Sie dort "Plugins" => "SlimBox2" markieren
  2. Speichern Sie dieses Preset im jQueryAdmin presets Unterverzeichnis (z. B. unter dem Namen "slimbox")
  3. Integrieren Sie folgenden Droplet-Aufruf z.B. in Ihr Template oder eine WYSIWYG-Sektion:
    [[jQueryInclude?preset=<Presetname>]]
    (Ersetzen sie <Presetname> durch den oben vergebenen Namen, also etwa "slimbox")
  4. Fügen Sie das rel="lightbox" Attribut bei den Links hinzu, die auf das Bild in Originalgröße verweisen
    Beispiel: <a href="images/image-1.jpg" rel="lightbox" title="my caption">image #1</a>
  5. Für Sets zusammengehörender Bilder, die Sie navigierbar machen möchten, fügen Sie einfach einen Gruppennamen gleich hinter "lightbox" hinzu:
    <a href="images/image-1.jpg" rel="lightbox-cats">image #1</a>
    <a href="images/image-2.jpg" rel="lightbox-cats">image #2</a>
    <a href="images/image-3.jpg" rel="lightbox-cats">image #3</a>
Für weitere Informationen besuchen Sie bitte die SlimBox2 Homepage.
